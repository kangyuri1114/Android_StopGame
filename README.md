## 기능
### 랜덤으로 제공되는 시간(초/밀리초)에 가장 가깝게 스탑워치를 멈춘 사람이 이기는 게임
- UI 구성 (버튼, 시간 표시 로직, 시작 - 게임 - 종료 화면)
- 목표 시간 랜덤으로 나타내기
- 타이머 구현
- 유한 반복 (참가자 수 제한하기)
- 각 참가자 점수 리스트에 저장, 출력
- 블라인드 모드 (랜덤 수가 보이지 않는) 구현
- 상태바 없애기
- viewBinding
